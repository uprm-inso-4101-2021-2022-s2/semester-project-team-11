Django Steps: (Open terminal in /DRF/)
- First Set Up:
    py -m venv venv
    Set-ExecutionPolicy Unrestricted -Scope Process     # Sometimes venv\Scripts\activate fails, this avoids it
    venv\Scripts\activate
    pip install django
    pip install djangorestframework
    pip install django-cors-headers
    pip install coverage                                # Testing library
    py manage.py makemigrations                         # Following two commands build the sqlite db
    py manage.py migrate
    py manage.py createsuperuser                        # Create an admin account (user=admin, password=password)
    py manage.py runserver                              # Run django backend

- Testing: # Postman app as a tool to see api
    coverage run --omit='*/venv/*' manage.py test       # Run tests and Omit venv folder from being tested
    coverage html                                       # Write index.html in htmlcov/, run it in a browser to see test results

- Shutting Down:
    CTRL + C                                            # Stops django server
    deactivate                                          # Deactivates virtual environment

- Starting Up After Shut Down
    Set-ExecutionPolicy Unrestricted -Scope Process     # Sometimes venv\Scripts\activate fails, this avoids it
    venv\Scripts\activate
    py manage.py runserver

- After making model changes or large modifications
    py manage.py makemigrations                         # Following two commands build the sqlite db
    py manage.py migrate
